مقدمة لأساليب التنقل في الأشجار الثنائية

عملية التنقل (Traversal) هي عملية المرور أو زيارة كل عقدة من عقد الشجرة مرة واحدة حسب ترتيب معين بهدف إجراء عملية معالجة ما عليها (مثل: طباعة، بحث، إضافة، حذف).

أنواع أساليب التنقل

يوجد أربعة أساليب رئيسية لتنقل الأشجار الثنائية:

Pre-order (VLR)

1. زيارة العقدة (V)

2. التنقل في الشجرة اليسرى (L)

3. التنقل في الشجرة اليمنى (R)

V → L → R
In-order (LVR)

1. التنقل في الشجرة اليسرى (L)

2. زيارة العقدة (V)

3. التنقل في الشجرة اليمنى (R)

L → V → R
Post-order (LRV)

1. التنقل في الشجرة اليسرى (L)

2. التنقل في الشجرة اليمنى (R)

3. زيارة العقدة (V)

L → R → V
Level-order

زيارة العقد مستوىً بمستوى من الأعلى إلى الأسفل، ومن اليسار إلى اليمين

المستوى 0 → 1 → 2 → ...

الشجرة المستخدمة في الأمثلة

مقارنة بين أساليب التنقل

النوع الاستخدامات التعقيد الزمني
Pre-order نسخ الشجرة، التعبير البوليني O(n)
In-order طباعة القيم مرتبة، التعبير الجبري O(n)
Post-order حذف الشجرة، التعبير البوليني العكسي O(n)
Level-order البحث بالعرض، حساب الارتفاع O(n)

التنقل بأسلوب In-order (LVR)

In-order (LVR): يبدأ التنقل بزيارة الابن الأيسر (L) ثم العقدة (V) ثم الابن الأيمن (R).

عملياً: يتم زيارة جميع العقد في الشجرة الفرعية اليسارية، ثم العقدة الحالية، ثم جميع العقد في الشجرة الفرعية اليمينية.

مثال توضيحي

التنفيذ باستخدام العودية (Recursive)

struct

Node {

    

int

data;

    

struct

Node* left;

    

struct

Node* right;

};


void

inOrder(

struct

Node* node) {

    

if

(node ==

NULL

)

return

;

    

    

// 1. التنقل في الشجرة اليسرى

    inOrder(node->left);

    

    

// 2. زيارة العقدة الحالية

    cout << node->data <<

" "

;

    

    

// 3. التنقل في الشجرة اليمنى

    inOrder(node->right);

}

التنقل بأسلوب Pre-order (VLR)

Pre-order (VLR): يبدأ التنقل بزيارة العقدة (V) ثم الابن الأيسر (L) ثم الابن الأيمن (R).

عملياً: يتم زيارة العقدة الحالية أولاً، ثم جميع العقد في الشجرة الفرعية اليسارية، ثم جميع العقد في الشجرة الفرعية اليمينية.

مثال توضيحي

التنفيذ باستخدام العودية (Recursive)

void

preOrder(

struct

Node* node) {

    

if

(node ==

NULL

)

return

;

    

    

// 1. زيارة العقدة الحالية

    cout << node->data <<

" "

;

    

    

// 2. التنقل في الشجرة اليسرى

    preOrder(node->left);

    

    

// 3. التنقل في الشجرة اليمنى

    preOrder(node->right);

}

التطبيقات العملية

  • نسخ الشجرة الثنائية
  • الحصول على التعبير البوليني من شجرة التعبير
  • إنشاء نسخة طبق الأصل من الشجرة
  • التخزين التسلسلي للشجرة

التنقل بأسلوب Post-order (LRV)

Post-order (LRV): يبدأ التنقل بزيارة الابن الأيسر (L) ثم الابن الأيمن (R) ثم العقدة (V).

عملياً: يتم زيارة جميع العقد في الشجرة الفرعية اليسارية، ثم جميع العقد في الشجرة الفرعية اليمينية، ثم العقدة الحالية.

مثال توضيحي

التنفيذ باستخدام العودية (Recursive)

void

postOrder(

struct

Node* node) {

    

if

(node ==

NULL

)

return

;

    

    

// 1. التنقل في الشجرة اليسرى

    postOrder(node->left);

    

    

// 2. التنقل في الشجرة اليمنى

    postOrder(node->right);

    

    

// 3. زيارة العقدة الحالية

    cout << node->data <<

" "

;

}

التطبيقات العملية

  • حذف الشجرة الثنائية
  • الحصول على التعبير البوليني العكسي
  • تحليل التعبيرات الرياضية
  • حساب حجم الذاكرة المستخدمة

التنقل بأسلوب Level-order

Level-order: في هذا الأسلوب من التنقل، يتم زيارة العقد في كل مستوى ابتداءً من العقدة الواقعة في أقصى اليسار إلى العقدة الواقعة في أقصى اليمين.

يتم زيارة جميع العقد في المستوى الحالي قبل الانتقال إلى المستوى التالي.

    queue<

while

(!q.empty()) {

        cout << node->data <<

" "

;

        q.pop();


        

// إضافة الابن الأيسر إذا موجود

        

if

(node->left !=

NULL

)

            q.push(node->left);


        

// إضافة الابن الأيمن إذا موجود

(node->right !=

NULL

    }

التطبيقات العملية

  • البحث بالعرض أولاً (BFS)
  • العثور على المستوى الأعرض في الشجرة
  • حساب عدد العقد في كل مستوى
  • طباعة الشجرة بشكل هرمي
